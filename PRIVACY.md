# 파치먼트 (Parchment) — 개인정보처리방침

**최종 업데이트:** 2026-05-13

파치먼트(이하 "서비스")는 사용자의 개인정보를 소중히 여기며, 관련 법령에 따라 안전하게 처리합니다. 본 방침은 서비스가 수집·이용하는 개인정보의 항목과 목적을 안내합니다.

## 1. 수집하는 정보

| 구분 | 항목 | 목적 |
|---|---|---|
| 회원가입(이메일) | 이메일, 비밀번호(해시), 표시 이름 | 계정 식별, 로그인 |
| 카카오 로그인 | 카카오 계정 식별자, 이메일 | OAuth 로그인 |
| Apple 로그인 | Apple ID 식별자, 이메일 (사용자 동의 시) | OAuth 로그인 |
| 익명 사용 | 익명 세션 ID | 북마크 동기화 |
| 위치 정보 | GPS 좌표 (앱 사용 중에만, 사용자 허용 시) | 주변 큐레이션 공간 추천 |
| 사진 | 사용자가 직접 첨부한 이미지 | 리뷰 작성 |
| 리뷰 | 작성자 이름, 본문, 평점 | 큐레이션 콘텐츠 표시 |

## 2. 보관 기간

- 회원 정보: 회원 탈퇴 즉시 삭제
- 익명 세션: 마지막 활동으로부터 12개월 후 자동 삭제
- 위치 정보: 서버에 저장하지 않음 (앱 메모리 내에서만 사용)

## 3. 제3자 제공

서비스는 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 아래 처리 위탁이 있습니다:

| 위탁사 | 위탁 항목 | 목적 |
|---|---|---|
| Supabase, Inc. | 인증/데이터베이스 | 클라우드 인프라 |
| Apple Inc. | Sign in with Apple | iOS 인증 위탁 |
| Kakao Corp. | 카카오 로그인 | OAuth 위탁 |
| OpenAI, L.L.C. | 챗봇 메시지 (1회성, 비저장) | 큐레이터 추천 응답 생성 |

## 4. 사용자 권리

- 본인의 개인정보 열람·수정·삭제를 요청할 수 있습니다.
- 회원 탈퇴: 앱 → 프로필 → 로그아웃 후 계정 삭제 요청을 아래 이메일로 보내주세요.

## 5. 안전 조치

- 비밀번호는 bcrypt 해시로 저장됩니다.
- 모든 통신은 HTTPS/TLS로 암호화됩니다.
- 서버 접근은 최소 권한 원칙을 따릅니다 (Row Level Security).
